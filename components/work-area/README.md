# Work Area Components

Work area (drawer panel) components in `components/work-area/`.

## Components

| Component | Purpose |
|-----------|---------|
| work-area-desktop.tsx | Desktop work area (side panel) |
| work-area-mobile.tsx | Mobile work area (bottom sheet) |
| work-area-content.tsx | Renders active drawer content |

## States

- **Desktop**: hidden, docked, fullscreen
- **Mobile**: closed, half, full

See [architecture/complete-architecture-documentation.md](../../architecture/complete-architecture-documentation.md#layout-and-components).
