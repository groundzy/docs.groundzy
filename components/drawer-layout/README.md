# Drawer Layout Components

Drawer layout primitives in `components/drawer-layout/`.

## Components

| Component | Purpose |
|-----------|---------|
| drawer-header.tsx | Drawer header with title, actions |
| drawer-body.tsx | Scrollable body |
| drawer-footer.tsx | Footer with actions |
| drawer-section.tsx | Section within drawer |
| drawer-form-section.tsx | Form section |
| filter-bar.tsx | Filter bar (search, filters) |
| list-container.tsx | Virtualized list container |

## Usage

Drawers use these primitives for consistent layout. See [features/drawer-system.md](../../features/drawer-system.md).
