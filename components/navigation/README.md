# Navigation Components

Navigation components in `components/navigation/`.

## Components

| Component | Purpose |
|-----------|---------|
| sidebar.tsx | Desktop sidebar (drawer list) |
| bottom-nav.tsx | Mobile bottom navigation |

## Behavior

- Drawer metadata from `getSidebarDrawerMetadata()`, `getBottomNavDrawerMetadata()`
- Tier-filtered visibility
- More drawer overflow on mobile
