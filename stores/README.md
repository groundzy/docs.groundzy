# Stores

Zustand stores in `stores/`.

## Stores

| Store | Purpose |
|-------|---------|
| navigation-store | Sidebar, work area, drawer history (persisted) |
| map-store | Map center, zoom, bounds, tool |
| selection-store | Selected tree, zone, property |
| draw-store | Draw mode, pending zone |
| measure-store | Measure mode, points, distance, area |
| tree-add-store | Add-tree form state |
| auth-store | Auth UI state |
| fast-mapping-store | Fast mapping mode |
| identifying-wand-store | AI identify wizard state |
| view-tree-header-store | View-tree header state |
| breakout-placement-store | Breakout placement |

See [reference/types-and-stores.md](../reference/types-and-stores.md) for details.
