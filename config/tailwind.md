# Tailwind Configuration

Tailwind CSS 4 configuration.

## Key Settings

- **Theme**: Custom colors, breakpoints in `tailwind.config.ts`
- **PostCSS**: `@tailwindcss/postcss`
- **Plugins**: tailwind-animate
- **Design tokens**: `lib/design-tokens.ts` (CSS variables)

## Breakpoints

See `lib/breakpoints.ts` for `isTabletOrUp()`, etc.
